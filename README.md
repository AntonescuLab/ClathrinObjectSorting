This is a Matlab function that is used to sort and analyze clathrin objects. 

This requires data to be first analyzed by cmeAnalysis, found at https://github.com/DanuserLab/cmeAnalysis and as described in https://www.cell.com/developmental-cell/fulltext/S1534-5807(13)00382-1 

This sortCLSdata function is used in https://www.cell.com/developmental-cell/fulltext/S1534-5807(13)00382-1.

Details for the use of sortCLSdata.m are available in a STAR Protocol paper planned for early 2026.
